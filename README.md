# 📌 qr-maker-react-ts

url, text를 입력해 qr code를 다운 받을 수 있는 대시보드 프로젝트입니다.

## 🙋‍♂️ 배포 주소

[qr-maker-react-ts](https://bofwio57.github.io/qr-maker-react-ts/)

## ⚙ 개발 환경

- React
- ts
- vite
- qrcode.react
- html-to-image
- lucide-react
- framer-motion
- tailwindcss

## 📝 기능

- url, text 입력시 실시간 qr code 생성
- png, jpg, svg로 qr code 저장
- localStorage를 이용해 히스토리 복구 가능
